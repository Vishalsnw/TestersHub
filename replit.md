# Overview
TestersHub is a native Android application built with Kotlin and Firebase to facilitate the exchange of Google Play closed testing support among developers.

# Project Architecture
- **Language**: Kotlin
- **Build System**: Gradle
- **Backend**: Firebase (Auth & Firestore)
- **UI**: XML Layouts with Material Design

# Features
- Google Sign-In via Firebase
- Real-time testing request feed
- Create and manage testing requests
- Join testing for other apps
- Progress tracking for testers required
- User profile with contribution stats

# Note
This project is generated as a standard Android Studio project structure. To build and run, open the root directory in Android Studio.
